# 📊 Customer Sales Analytics Dashboard

An interactive **Customer Sales Analytics Dashboard** built using **Power BI** to analyze sales performance, customer behavior, and business insights. This dashboard helps businesses make data-driven decisions by visualizing key sales metrics, customer trends, and product performance.

---

## 🚀 Project Overview

This project provides a comprehensive analysis of customer sales data through interactive dashboards. It enables users to:

- Monitor overall sales performance
- Analyze customer purchasing behavior
- Identify top-performing products
- Track regional sales performance
- Measure profit and revenue trends
- Support strategic business decision-making

---

## 📌 Features

- 📈 Sales Overview Dashboard
- 👥 Customer Analysis
- 💰 Revenue & Profit Analysis
- 🛍️ Product Performance
- 🌍 Regional Sales Insights
- 📅 Time-Based Sales Trends
- 🔍 Interactive Filters & Slicers
- 📊 KPI Cards for Quick Insights
